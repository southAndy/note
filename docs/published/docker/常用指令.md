### 容器操作

```shell
# 列出運行中的容器
docker-compose ls

# 檢視容器（包含 stop 的）
docker-compose ls -al

# 以 yml 建立容器指令
docker-compose build

# 刪除容器
docker-compose down [container-id]

# 進入容器的 shell
docker-copmose exec -it [container-id] sh
```
