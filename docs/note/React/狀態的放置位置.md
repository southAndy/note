
#react 

狀態何時應該寫在元件內即可，而不用升上去

```
// 1. 只有該組件及其直接子組件使用
const [isShowMemberMenu, setMemberMenu] = useState(false) // ✅ 只有 User 和 UserMenu 用到

// 2. 組件的內部交互狀態
const [isHovered, setIsHovered] = useState(false) // ✅ hover 效果
const [activeTab, setActiveTab] = useState(0) // ✅ tab 切換

// 3. 臨時的 UI 狀態
const [isExpanded, setIsExpanded] = useState(false) // ✅ 展開/收合
```


需要上升的狀態

```
// 1. 多個兄弟組件需要共享
const [searchQuery, setSearchQuery] = useState('') // 🔄 SearchInput 和 SearchResults 都需要

// 2. 父組件需要控制子組件
const [currentStep, setCurrentStep] = useState(1) // 🔄 多步驟表單

// 3. 狀態影響其他組件的行為
const [isModalOpen, setIsModalOpen] = useState(false) // 🔄 會影響背景遮罩、滾動鎖定等
```
