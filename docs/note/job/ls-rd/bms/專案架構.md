#bms


## 架構

```
src
- asset
	- javascripts //共用邏輯、配置檔案、API設定、"前端全域資料庫(ex:vuex)"
	- styles //專案全域等級的樣式設定，如 color.scss
	- images 
- components //共用元件
	- table //專屬 table 的元件
- containers
- layout
- views //頁面檔
- router
- store
```

### 自我釐清
> views 跟 containers 差在哪
