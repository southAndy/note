### 解決跨結構、多層級的元件資料傳遞問題：

目前前端主流的框架最基本單位就是元件，在vuex還沒出現以前，如果想要實現「==跨元件的資料如何傳遞==」，以vue來說，就是使用`props / @emit ` 或是 `provide / inject`  來進行。

```html

//假設今天有一個頁面的元件架構如下：
<Home>
	<Child>
		<GrandChild/>
			<GrandGrandChild/>
</Home>
```
如上面的例子，當你的頁面架構層級之多，從實際開發面來說，這樣容易產生的問題有：
	1. **在後續追蹤功能上成本較高**
	2. 較難保證在過程不被其他內容影響
