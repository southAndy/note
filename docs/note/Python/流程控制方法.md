#python 

最常見的：
1. `if else`
2. 三元判斷
3. 多個條件同時成立時 `and`
4. 其一成立即可 `or`
5. 不是時成立 `not`

```python
# if .. else
data = true
if data:
   print('is true!')
else: 
   print('not! is false')
   
# 三元判斷



# and
if a and b:
   print('both success!')

# or
if a or b:
   print('one is true!')

# not
if not a:
   print('not !')
```
