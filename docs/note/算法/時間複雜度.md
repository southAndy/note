
**時間複雜度**（Time Complexity）用來描述一個演算法在執行時，所需時間（或執行步驟）隨著**輸入資料大小**（通常以 n 表示）的變化而成長的情況。它告訴我們：**當輸入規模越來越大時，演算法的執行時間會如何變化**。

```js
for (let i = 0; i < n; i++) {
    for (let j = 0; j < n; j++) {
        // 這裡的操作會執行 n * n 次
    }
}
```

以這個案例來說，外層迴圈執行 n 次，內層迴圈每次也執行 n 次，總計 n \times n = n^2。

所以 **時間複雜度：O(n^2)。**
